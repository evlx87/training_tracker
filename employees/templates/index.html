ℹ️ Инструкция по работе с порталом

Система "Training Tracker" предназначена для управления данными сотрудников, их обучением, подразделениями и должностями. Вот краткое руководство для начала работы:

  1. Список сотрудников: Просматривайте, добавляйте, редактируйте или удаляйте данные сотрудников и их записи об обучении.
  2. Структурные подразделения: Управляйте подразделениями, создавайте новые или редактируйте существующие.
  3. Программы обучения: Создавайте программы, указывайте периодичность и отслеживайте прогресс.
  4. Должности: Управляйте списком должностей в организации.
  5. Отчеты по обучению: Формируйте отчеты с фильтрацией и экспортом в Excel.
  6. Удаление данных: Для удаления записей требуется подтверждение от пользователей группы MTO.

Используйте кнопки в верхней части страницы для навигации по разделам.
